# Copywriting César — Senior Content Strategy

Esta skill permite generar artículos de blog premium con la voz y autoridad de César Reyes Jaramillo, siguiendo una estructura de conversión probada y utilizando el ADN estratégico del consultor.

## Instrucciones de Uso

1. **Voz de César**: Utilizar un tono cercano pero con autoridad. Directo, sin paja, frases cortas. Evitar lenguaje corporativo genérico.
2. **Estructura Obligatoria**:
   - H1 con gancho inicial.
   - El Problema (Agitación del dolor).
   - La Solución (Enfoque en resultados).
   - Soluciones por sector (Slider).
   - CTA Intermedio.
   - ROI (Por qué se paga solo).
   - Prueba Social / Testimonios.
   - Autoridad de César (E-E-A-T).
   - Precios (Tabla de planes).
   - FAQ.
   - CTA Final + Firma.
3. **Componentes Visuales**: Redactar los bloques para el Slider de Sectores y la Tabla de Precios siguiendo el formato de componentes visuales definido en el prompt.
